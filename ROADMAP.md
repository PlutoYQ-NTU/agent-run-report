# Roadmap

`agent-run-report` starts intentionally small: a local CLI that creates readable Markdown and JSON reports after coding-agent runs.

Future ideas:

- GitHub PR comment mode.
- Multiple log inputs for long-running or split agent sessions.
- HTML output.
- Richer test framework detection beyond simple text hints.
- Integration with agent approval tools.
- GitHub Actions support.
- Config files for project-specific report templates.
- Optional redaction helpers for common secret patterns.
