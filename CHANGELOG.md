# Changelog

All notable changes to this project will be documented in this file.

## v0.1.1 - Unreleased

### Added

- Added GitHub Actions CI.
- Added issue templates.
- Added README badges.
- Added related-project links.

## v0.1.0 - 2026-06-01

### Added

- Initial `agent-run-report` CLI.
- Markdown and JSON report output.
- Git repository status, changed file, diff stat, and optional diff excerpt collection.
- Rule-based agent log parsing for errors, warnings, tracebacks, failed command hints, and test summaries.
- Suggested next-step generation for common maintainer workflows.
- Example input and output files.
- Unit tests for parser, report model, and Markdown writer.
